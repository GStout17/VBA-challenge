# VBA-challenge
The VBA of Wall Street

Note:

*This VBA script has the capability of searching through multiple worksheets while transforming large amounts of data into an accessible format. 

*Information gathered included ticker symbol, % changed, yearly change, and total volume of each stock.


Details of Script:

* Defined variables to search through each worksheet in excel file by determining the last row and creating a loop.
* Added headers to each column that included “Ticker, yearly change, percent change, total stock volume”.
* Set several variables to check for the above-mentioned categories via a loop.
* Created conditional formatting to yearly changes in each worksheet. This was done by another loop function. 
*additional loop and column values to assess “greatest % increase, greatest 5 decrease. Greatest total volume, ticker, and value”
